This directory has sample file programs. The lisiting and explanation is as below:


1. file-copy.c
---------------
Copies the contents from one file to another. Source file used: input-file.txt


2. lists-and-files.c
---------------------
Reads the data from file to list. Operates on list. In the end, new updated list data is dumped back to file before the program terminates. Source file used: program-data.txt


3. mobile-service-provider.c
-----------------------------
Simulates the program of a mobile service provider using file operations. Program is well commented. Source file used: service.txt
